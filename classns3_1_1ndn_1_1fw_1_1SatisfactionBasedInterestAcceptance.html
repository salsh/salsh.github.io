Detailed Description

template<class Parent>
class ns3::ndn::fw::SatisfactionBasedInterestAcceptance< Parent >

Strategy implementing stats-based randomized accept for interests.

This strategy has several limitation to accept interest for further processing:

  • if per-fib-entry limit or per-face limit is exhausted, Interest will not be accepted
  • if the number of interests received from the incoming face is less than threshold, then no special handling
  • if this number is greater than threshold, an Interest will be accepted with probability equal to satisfaction ratio for this incoming face (overall per-face). (probability is shifted to allow small rate of acceptance (1% by default) of Interests from faces with 0 satisfaction ratio.
